# Y-Maze Tracker Web App

This repository contains the **source code** for the Y-Maze Tracker web application.
You are free to use, modify, and distribute this code under the terms of the included license.


---
## Features

* Record arm entries (A, B, C) during Y-Maze sessions
* Automatically compute alternation rates and related metrics
* Adjustable session timer
* Export session data as CSV
* Works offline (Progressive Web App support)
* Store sessions locally on your device

---

## Installation on mobile


This app can be **installed on your smartphone** as a Progressive Web App (PWA).
To install it:


you also can just open the website once in your mobile device and until you remove your history and cookies the site will remain accessible offline

1. Open the website in your mobile browser.
2. Tap the **Share** button (or menu icon).
3. Choose **“Add to Home Screen”**.
4. The app will then appear as a standalone icon and can be used **offline** after the first load.
