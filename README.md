## Features

* Record arm entries (A, B, C) during Y-Maze sessions
* Automatically compute alternation rates and related metrics
* Adjustable session timer
* Export session data as CSV
* Works offline (Progressive Web App support)
* Store sessions locally on your device

## Installation on mobile

This app can be **installed on your smartphone** as a Progressive Web App (PWA).
To install it:

1. Open [https://ymaze-tracker-pwa.vercel.app/](https://ymaze-tracker-pwa.vercel.app/) in your mobile browser.
2. Tap the **Share** button (or menu icon).
